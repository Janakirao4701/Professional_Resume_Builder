# Project - Resume Builder

## Overview
A unified single-page application for candidate resume generation, editing, and layout customization.

## Business Goals
* Enable users/recruiters to seamlessly switch between multiple candidate profiles from a single UI.
* Allow dynamic creation, editing, and deletion of custom candidate profiles directly from the browser.
* Generate and package resumes into standard DOCX format using clean custom styles and alignments.

## Tech Stack
* **Structure**: HTML5 (Semantic tags, dialog structures)
* **Styling**: Vanilla CSS3 (Custom fonts, margins, CSS grid, flex layouts)
* **Logic**: Vanilla JavaScript ES6 (Proxy state binding, localStorage caching)
* **Libraries**: docx.js (v7.3.0 loaded via JSDelivr CDN)
